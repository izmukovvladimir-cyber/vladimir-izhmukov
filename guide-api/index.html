Что это и зачем тебе

Бесплатный доступ к живым данным без карты и подписок.

API это способ, которым одна программа берёт данные у другой. Курс валют, счёт матча, свежие новости, вакансии, погода. Обычно за такие данные берут деньги. В этом репозитории собраны источники, которые отдают данные бесплатно. Ты подключаешь нужный источник к своему приложению или автоматизации, и оно само тянет актуальную информацию, тебе не надо вводить её руками.

Открыть каталог бесплатных API на GitHub

Это открытый каталог, обновляется сообществом. Сверху есть оглавление по категориям. Жми на нужную категорию, и попадаешь в таблицу: название сервиса, что он отдаёт, нужен ли ключ, есть ли бесплатный доступ. Начни с категории, близкой к твоей задаче.

Как быстро найти нужный API

Три шага, чтобы не утонуть в списке.

  1. Открой оглавление сверху страницы и выбери категорию под свою задачу: Finance, News, Jobs, Sports, Weather и так далее.
  2. В таблице смотри колонку Auth. Значение No означает, что ключ не нужен и можно дёргать данные сразу. apiKey означает, что нужен бесплатный ключ, его выдают за минуту на сайте сервиса.
  3. Открой ссылку на выбранный сервис, найди раздел документации со словом Endpoints или Reference. Там показано, по какому адресу запрашивать данные.

Живой пример

Как это работает на понятной задаче.

Допустим, ты собираешь простое приложение со счётом матчей в реальном времени. Идёшь в категорию Sports, выбираешь спортивный API с бесплатным доступом, берёшь адрес его endpoint и подключаешь к своему приложению. Приложение по этому адресу запрашивает данные и мгновенно показывает актуальный счёт. Точно так же подключается погода, новости, курс валют или вакансии, меняется только выбранный источник.
