Omma: 3д-сайты, приложения и игры из одного промпта

Обычно за интерактивный 3д-сайт или мини-игру платят разработчику и ждут недели. Omma собирает это из одного текстового промпта: ты описываешь словами, а несколько ИИ-агентов работают параллельно, один пишет код, второй строит 3д-модели, третий рисует картинки. Движок работает на WebGPU, поэтому физика, отражения и анимация крутятся прямо в браузере в реальном времени. Ниже прямая ссылка, три стартовых промпта и приём, как превратить это в wow-ролик для блога.

Разбор Omma: что это, прямая ссылка, 3 стартовых промпта под сцену, сайт и мини-игру, плюс приём для контента. Подстановки в [квадратных скобках].

Как пользоваться: открой omma.build по кнопке ниже, старт без карты. Опиши, что хочешь собрать, обычными словами, можно по-английски для лучшего результата, но русский тоже понимает. Смотри, как агенты собирают сцену, потом докручивай правками: «сделай темнее», «добавь дождь», «замени шрифт». Каждая правка это просто следующее сообщение.

Стартовая 3д-сцена под твой бренд

Промпт
Create an interactive 3D hero scene for a personal brand website. Theme: [твоя ниша, например growth on social media]. A floating 3D [объект по теме, например smartphone with a growing chart bursting out of the screen], dark premium background, accent color [твой цвет], soft studio lighting, slow rotation, the object reacts to mouse movement. Add a bold headline [твоя фраза] and one button [текст кнопки].

Пример: объект: smartphone with rocket, neon wireframe globe, glowing brain.

Одностраничник с 3д-героем

Промпт
Build a one-page website for [твой продукт, например a 45-day Reels challenge]. Structure: hero with an interactive 3D scene on theme [тема], then 3 benefit cards, one testimonial block, and a final call-to-action button [текст кнопки]. Мини-игра для вовлечения

Промпт
Make a simple browser mini-game: [механика, например catch falling coins with a basket, 30 seconds timer, score counter]. Style it for my brand: background [цвет или настроение], the falling objects are [объекты по твоей теме], show a final screen with the score and a message [твоя фраза с призывом]. Keep controls to one finger so it works on a phone.

Пример: механика: лови объекты, прыгай через препятствия, собирай пары.

4 из 4

Wow-ролик для блога из твоей сцены

Промпт
Готовую сцену преврати в контент: включи запись экрана на телефоне или компьютере, медленно двигай мышью по 3д-сцене, покрути объект, покажи физику. Этот футаж и есть твой ролик: сверху хук «Я собрал это без единой строчки кода за 5 минут», дальше 15 секунд самой сцены, в конце призыв с твоим кодовым словом. Такой формат собирает комментарии «как?» сам по себе, потому что выглядит как работа студии.

Пример: это не промпт для нейросети, это шаг для тебя: сцена уже готова, осталось снять экран.

Что такое Omma

Коротко, чтобы понимать, чем это отличается от обычных ИИ-сборщиков сайтов.

  • Сделан командой Spline, главного инструмента веб-3д. Запустился весной 2026.
  • Вместо одного медленного ИИ работает команда агентов параллельно: код, 3д-модели, картинки.
  • Работает на WebGPU: физика, отражения и симуляции идут прямо в браузере, без установки программ.
  • Собирает сайты, приложения, игры, презентации и 3д-сцены из текстового описания.
  • Старт без оплаты и без карты, платный тариф нужен только на объёмах.
