6 навыков Claude в один контент-цех: установка и связка

Это пошаговый гайд по установке шести навыков Claude, которые превращают его из чат-помощника в контент-цех: дизайн-мышление, готовые скриншоты витрины, настоящие презентации, система команд по этапам работы, оформленные PDF и генератор собственных навыков. По каждому навыку ниже лежит, что он реально делает, точная команда установки в поле для копирования и живой пример применения под контент или блог. В конце разложена связка: в каком порядке звать эти навыки в один рабочий день, чтобы один планировал, три производили, один проверял, а последний зашивал удачную схему на повтор. Прочитал только эту страницу, вставил шесть команд, и у Claude появился рабочий контент-цех. Кодить руками не нужно.

Читается за 9 минут. Внутри установка 6 навыков Claude с точными командами, что делает каждый, пример применения под контент по каждому и финальная схема, как связать их в один конвейер рабочего дня.

Ставь по одному. Всё сразу осваивать смысла нет. Открой Claude Code, скопируй команду из блока под нужным навыком, вставь в строку и отправь. Три навыка (PPTX, PDF, Skill Creator) живут в одном официальном наборе anthropics/skills: добавляешь набор один раз, а дальше просто включаешь нужный навык по имени. После установки проверь навык на своей реальной задаче из примера, чтобы сразу увидеть результат. Дочитаешь до конца, собери из них конвейер по последней секции.

Перед стартом: что тебе понадобится

Все шесть навыков ставятся в Claude Code, это среда, где Claude умеет выполнять команды и работать с файлами. Команды бывают двух видов. Первый вид начинается со слэша, например /plugin, его вставляешь прямо в строку Claude Code. Второй вид это обычные консольные команды (npx, git clone), их запускаешь в терминале проекта. Ниже под каждым навыком помечено, какой это случай. Три навыка из шести идут из одного официального набора Anthropic, поэтому его добавляешь один раз, а потом включаешь по частям.

Официальный набор anthropics/skills содержит сразу три навыка из этого гайда: PPTX, PDF и Skill Creator. Добавь набор одной командой, а дальше включай нужный навык по имени. Отдельно скачивать его три раза не нужно.
Промпт
/plugin marketplace add anthropics/skills

1. UI-UX Pro Max: дизайн-мышление для Claude

Навык загружает в Claude сотни правил дизайна, готовых стилей и палитр плюс генератор дизайн-систем по брифу. После установки Claude перестаёт выдавать случайные цвета и отступы и начинает думать как дизайнер: собирает под твой бренд цельную систему, где цвета, шрифты и отступы связаны между собой. Команда установки начинается со слэша, вставляй её прямо в Claude Code.

Промпт
/plugin marketplace add nextlevelbuilder/ui-ux-pro-max-skill
Пример под контент. Дай бриф: «Собери дизайн-систему под мой лид-магнит: ниша, настроение премиум и доверие, база чёрный и лаймовый акцент. Дай палитру из 5 цветов с кодами, пару шрифтов, размеры заголовков и отступы.» Получаешь готовую систему, по которой карусели, обложки и страницы хаба держат один узнаваемый стиль.

2. App Store Design: готовые скриншоты витрины

Навык собирает готовые к публикации скриншоты для витрины App Store и Play Store сразу во всех размерах Apple и Android. Если запускаешь мини-приложение, бота с витриной или продукт со страницей в сторе, он снимает с тебя ручную возню с размерами и рамками. Это консольная команда, запускай её в терминале своего проекта.

Промпт
npx skills add ParthJadhav/app-store-screenshots
Пример под контент. Если у тебя есть мини-продукт или приложение, попроси собрать комплект витринных скриншотов с заголовками-выгодами под каждый экран в размерах iPhone и Android разом. Один заход, и превью для стора и для поста-анонса готовы без дизайнера.

3. PPTX: настоящие презентации

Навык делает настоящие файлы .pptx, которые открываются в Keynote, PowerPoint и Google Slides и остаются редактируемыми. Не картинки слайдов, а живой файл, где можно поправить текст. Навык входит в официальный набор anthropics/skills, поэтому сначала добавь набор командой из блока перед стартом, потом включи навык по имени pptx.

Промпт
/plugin marketplace add anthropics/skills
После добавления набора включи навык pptx (по имени) и держи его в проекте. Дальше проси собрать презентацию текстом задачи.
Пример под контент. «Собери презентацию на 8 слайдов для мини-урока по нейроописаниям: обложка, боль аудитории, метод по шагам, пример, призыв.» Получаешь .pptx, открываешь в Keynote, записываешь экраном и выкладываешь как видео-урок или лид-магнит.

4. GStack: 23 команды по этапам работы

Навык добавляет 23 навыка-команды, разложенные по этапам работы: Think, Plan, Build, Review, Ship, Reflect. Это готовый рабочий процесс для Claude от идеи до публикации. Внутри есть команда /plan-design-review, которая оценивает дизайн по шкале от 0 до 10 и говорит, что подтянуть. Это консольная установка через git, запускай в терминале проекта.

Промпт
git clone github.com/garrytan/gstack.git && ./setup
Пример под контент. Перед публикацией прогони макет карусели или лендинг через /plan-design-review. Получаешь оценку дизайна от 0 до 10 и конкретный список правок, и правишь до выхода, пока пост ещё не опубликован.

5. PDF: оформленные лид-магниты

Навык делает настоящие оформленные PDF: лид-магниты, паки промптов, чек-листы, свайп-файлы. То есть сам файл, который ты отдаёшь человеку по кодовому слову. Верстать его где-то ещё не придётся. Навык тоже живёт в наборе anthropics/skills, добавь набор командой из блока перед стартом и включи навык по имени pdf.

Промпт
/plugin marketplace add anthropics/skills
После добавления набора включи навык pdf. Дальше проси собрать документ текстом задачи.
Пример под контент. «Собери PDF-пак на 5 страниц: 10 промптов для хуков с полем для копирования, оформи под мой бренд, титульная страница с выгодой.» Получаешь готовый файл, который сразу идёт в воронку как лид-магнит за подписку или кодовое слово.

6. Skill Creator: свой навык из любого процесса

Навык превращает любой твой повторяющийся процесс в собственный навык Claude. Один раз показываешь, как ты делаешь задачу, и Skill Creator упаковывает это в многоразовый навык, который дальше зовёшь одной командой вместо длинного объяснения каждый раз. Входит в набор anthropics/skills, добавь набор и включи навык по имени skill-creator.

Промпт
/plugin marketplace add anthropics/skills
После добавления набора включи навык skill-creator. Дальше опиши процесс, который хочешь зашить в навык.
Пример под контент. Опиши свой путь сборки карусели по референсу: как разбираешь чужой слайд, как переносишь смысл на свою нишу, как оформляешь текст. Skill Creator зашьёт это в навык, и завтра ты соберёшь карусель одной командой, не объясняя Claude процесс заново.

Как связать шесть навыков в один конвейер

По отдельности каждый навык полезен, но сила в связке. Вот реальный порядок использования за один рабочий день контент-мейкера: GStack держит процесс от плана до выпуска, UI-UX Pro Max, PPTX, PDF и App Store Design производят материалы в едином стиле, Skill Creator в конце зашивает удачную схему на повтор. Пройди по шагам.

  1. Планируй через GStack. Открой день командами этапа Think и Plan: распиши, что выпускаешь сегодня, какие форматы и под какую цель. GStack задаёт рамку, чтобы работа шла от идеи к публикации по порядку.
  2. Задай дизайн-систему через UI-UX Pro Max. Один раз собери палитру, шрифты и отступы под бренд. Дальше всё, что производишь за день, держит один стиль, и контент выглядит как единая система.
  3. Производи материалы. Презентацию-урок собирай навыком PPTX, сам лид-магнит навыком PDF, витрину продукта навыком App Store Design. Все три опираются на дизайн-систему из прошлого шага, поэтому выходят в одном узнаваемом виде.
  4. Проверь перед выпуском через GStack. Прогони готовое через /plan-design-review, получи оценку от 0 до 10 и список правок. Этапы Review и Ship закрывают публикацию только после того, как дизайн дотянут до планки.
  5. Зашей повтор через Skill Creator. Удачную дневную связку (например, сборку лид-магнита из PDF под свой бренд) упакуй в собственный навык. Завтра ты запускаешь её одной командой, и этап Reflect из GStack превращается в рост скорости на каждой следующей задаче.
Правило конвейера: планирует один навык (GStack), стиль задаёт один (UI-UX Pro Max), производят три (PPTX, PDF, App Store Design), а Skill Creator превращает удачный день в многоразовый навык. Собери связку один раз, и дальше контент-цех работает по кругу, ускоряясь с каждой итерацией.
